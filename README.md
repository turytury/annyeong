Annyeong
================
Product Management Project

### Prerequisites
- Java
- Maven
- MySQL
- myBatis

### Running Project
Clone project from repository
```
git@github.com:turytury/annyeong.git
```
Build project
```
mvn clean install
```
Run project
```bash
mvn spring-boot:run
```
