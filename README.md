Annyeong
================
Product Management Project

### Technology
- Java
- Maven
- SpringBoot
- Bootstrap
- MySQL
- MyBatis

### Running Project
Clone project from repository
```
git@github.com:turytury/annyeong.git
```
Build project
```
mvn clean install
```
Run project
```bash
mvn spring-boot:run
```

### Main Function
```bash
Create, Read, Update, Delete, Search, Sort category and product.
```
